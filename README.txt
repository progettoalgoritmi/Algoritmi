Guida comandi GIT
1. entrate su gitbash ed effettuate il pull dei file così da avere i file aggiornati all'ultima versione corrente
2. effettuare le modifiche ai file (blablabla buon divertimento)
3. inserire in gitbash la stringa "git add nome_file"  //in alternativa basta scrivere "." senza virgolette per modificare TUTTI i file
4. inserire la stringa "git commit -m 'commento' " //apici singoli compresi e vedere che sputa (cit.Antonio)
5. inserire la stringa "git push -u origin master" //Aggiorna effettivamente i file sul sito
